# Warehouse-Stocktifity
ISPM FINAL PROJECT - Warehouse Management System 


### How to run

```
git clone https://github.com/whend12/Warehouse-Stocktifity.git
```


#### RUN BACKEND
```
cd backend-Stocktifity 
```
```
npm i 
```

```
nodemon index
```



#### RUN FRONTEND
```
cd frontend-Stocktifity 
```
```
npm i 
```
```
npm start
```
